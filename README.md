# M2M_KeypadDisplay_Python

Python module for handling the [M2M Keypad/Display module](https://github.com/m2m-solutions/M2M_KeypadDisplay_PCB).

## Default pin mapping

### Button pins

pin|Usage
---|-----------
17 | Button A
18 | Button B
27 | Button Left
22 | Button Up
23 | Button Down
24 | Button right
25 | Button C

### LED pins

pin|Usage
---|----------
16 | Red LED
20 | Green LED
21 | Yellow LED
